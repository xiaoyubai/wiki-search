About Wiki-search

Wiki-search is a categorical search engine for wikipedia, user input includes both keyword and category.

When searching wikipedia, there are sometimes muliple pages related to the same keyword. Matrix factorization model is used to categorize all english wikipedia pages. Only results in the same category as user input will be considered for ranking purposes.
